----------ソフトの使い方について----------
基本、このソフトは音声Mod以外は対応していません。希望があれば追加します。
音声Mod一覧から音声を選択するか、フォルダを選択で導入したい音声Modを選択してModのダウンロード/インストールを押すと導入されます。
BGMMod一覧画面を押すとBGMModが導入できます。
サンプル音声では音声Mod一覧で指定された音声の一部が再生されます。
下にあるバーではサンプル音声の音量を指定できます。
キャッシュを削除ではソフト上で生成されたファイル又はダウンロードされたファイルを削除します。

----------BGMMod一覧について-----------
この機能を使用するには、必ず音声Modを導入した後に行う必要があります。
理由として、汎用性を持たせるために.yamlを上書きするのではなく、内容を変更するようにしているためです。
BGMModは、戦闘開始の音声を犠牲にして再生しているため、戦闘開始時に音声は流れなくなります。
そのため、戦闘開始の音声を戦闘開始前のタイマーで流れるようにしています。
BGMModは、Music/Music/Musicの構成以外では動かないため一覧以外のModは指定できません。
BGMMod作成ボタンでは/Resources/Musicフォルダ内にあるmp3ファイルを取得してMusic/Music/Musicの構成を維持したままFSBファイルを作成します。
BGMModはファイル容量が大きくなる可能性が高いため、10個以上音楽は入れれないようにしています。(手動で追加すれば制限はないです。)
また、10個以上のファイルが見つかった場合、ランダムでファイルが選択されます。
BGMファイルがMP3ファイルでない場合(.wavファイルなど)はMP3に自動で変換しますが、一回押すごとに一ファイルしか変換できないため、エラーが起こらないまでボタンを押し続ける必要があります。
できたデータは/Resources/BGMMod内に入っています。

----------FSBファイルの機能について--------
右上に2つFSBに関するボタンが設置されています。上のほうのボタンでは、文字どうりFSBファイルの中身をMP3へ変換させるものです。
FSBファイルがDVPL化されている場合、エラーが起こりますので、dvplを解除してからお試しください。
変換するfsbの中身が多い場合や1つ1つの音源の容量が大きい場合は変換するのに多少時間がかかります。そのため、完了を知らせるログが出るまで操作しないことをお勧めします。
下のボタンは上のボタンを実行すると機能します。(必要なファイルがない場合はダウンロードされます。)
下のボタンを実行すると、自動でプロジェクトファイルが生成され、それをダウンロードされたFMOD Studio API Windowsで開きます。
(FSBファイルによって名前順で指定されていないもの(私の音声Modがほとんどこれです)があってプロジェクトデータを生成させるのにかなり苦労しました。。。)
FSBのファイル名はソフト側で指定できるようにしてもよかったですが、レイアウトがわかりにくくなるためFMOD Studio API Windowsで指定してください。(デフォルトでingame_voice_ja.fsbにしています。)
また、圧縮方法も手動で設定します。(デフォルトでMP3のクオリティ75に設定しています。)
ビルドすると、実行ファイルがあるフォルダ内の/Resources内に作られます。(手動で設定した場合を除く)

---------音声、BGM共有の仕様----------
FMOD Studio API Windowsが開くと、実行フォルダ内の/Resources/Projects/内にVoice_Mod.fsproj又はBGM_Mod.fsprojが生成されているので用途に合ったプロジェクトをロードしてビルドします。
最初のほうにも書いていますが、基本音声Mod以外には対応していません。何か要望があればこれから追加していこうと思います。
音声、BGMModのプロジェクトデータはプログラムですべて自動化しているため、正常に動作しない可能性があるかもしれません。それに関してはModの内容によって修正の手順が異なるため各自で修正するようお願いします。

----------バックアップの仕様----------
バックアップファイルは、このソフトが初めて起動された、又はバックアップファイルが手動で削除されて、次にこのソフトが開かれた時のものです。
このソフトが起動される前にWoTBのファイルの変更を行っていた場合、変更されているファイルをバックアップファイルとして認識しますのでご注意ください。
また、バックアップファイルは音声Modに関するファイルしか保存されないので、砲撃音や照準Modは対応されていません。
